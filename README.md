# AtCoder

## 概要

[kabik](https://atcoder.jp/users/kabik) がAtCoderの問題を解く時に使っているリポジトリです。

過去に解いた問題の回答やメモを保存してます。


## コンテスト準備

`abc{contest number}` というディレクトリを作り、`template.py` の内容を `abc{contest number}/*.py` にコピーします

```bash
./python_dir.sh {contest number}
```
